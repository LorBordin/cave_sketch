# CaveSketch — Documentazione Web App

🌍 Lingue disponibili: [🇬🇧 English](README.md) | [🇮🇹 Italiano](README.it.md)

CaveSketch è un'applicazione web basata su [Streamlit](https://streamlit.io/) che trasforma le esportazioni `.dxf` di [TopoDroid](https://sites.google.com/site/spabordin/) in rilievi speleologici professionali e visualizzazioni geospaziali — senza bisogno di software GIS pesanti.

🔗 **Provalo subito:** [CaveSketch Live App](https://cavesketch.streamlit.app/)

---

## 📖 Guide

- **[Unione rilievi](survey-merging.it.md)** — Unisci un rilievo padre e uno figlio in un unico PDF, con tre protocolli per la sezione (Semplice, Specchio, Dislocamento).
- **[Mappe satellitari ed esportazione](satellite-maps.it.md)** — Posiziona i rilievi su immagini satellitari, combina più rilievi su un'unica mappa ed esporta in HTML, KML o KMZ offline.
