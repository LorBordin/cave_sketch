# CaveSketch — Web App Documentation

🌍 Available languages: [🇬🇧 English](README.md) | [🇮🇹 Italiano](README.it.md)

CaveSketch is a browser-based application built with [Streamlit](https://streamlit.io/) that turns [TopoDroid](https://sites.google.com/site/spabordin/) `.dxf` exports into professional cave survey plots and geospatial visualizations — no heavy GIS software required.

🔗 **Try it now:** [CaveSketch Live App](https://cavesketch.streamlit.app/)

---

## 📖 Guides

- **[Survey Merging](survey-merging.md)** — Combine a parent and child survey into a single PDF plot, with three section protocols (Simple, Mirror, Displacement).
- **[Satellite Maps & Export](satellite-maps.md)** — Place cave surveys on satellite imagery, combine multiple surveys on one map, and export to HTML, KML, or offline KMZ.
